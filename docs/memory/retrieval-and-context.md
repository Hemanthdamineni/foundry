# Retrieval and Context

Advanced retrieval is deferred for MVP.

The MVP runtime validates one feature workflow through `submit_output`,
ToolExecutor, ToolGate, SQLite, checkpoint, and recovery. It does not require
context harvesting, dependency expansion, semantic memory, vector search, or
cross-task learning.

Existing retrieval/indexing files are reference material only until they are
explicitly promoted after the core loop is operational.
